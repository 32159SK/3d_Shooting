
  ○AQUA GAME LIBRARY 7.17

    [概要]
      DXライブラリを使用したWindows用ゲーム制作ライブラリ
      ・ゲームループの管理
      ・ゲームオブジェクトの木構造化
      ・グラフィックス処理(スプライト・アニメーション付きスプライト・ラベル・モデル・カメラ・ビデオ)
      ・入力処理(キーボード、マウス、ジョイパッド)
      ・サウンド処理(SE・BGM)
      ・デバッグ機能
      ・その他ユーティリティ機能
